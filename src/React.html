1. Introdução ao React

O que é React e por que utilizá-lo:

React é uma biblioteca JavaScript utilizada para construir interfaces de usuário (UI) interativas. É eficiente pois atualiza apenas partes específicas da UI quando o estado dos dados muda, ao invés de renderizar a página inteira. Aqui está um exemplo básico:

Exemplo simples de um componente React:

import React from 'react';

function App() {

return (

Olá, React!

Bem-vindo ao meu primeiro aplicativo React.

); }

export default App;

Configuração do ambiente (Create React App):

O Create React App é uma ferramenta oficial para iniciar rapidamente um projeto React. Comandos como npx create-react-app my-app configuram a estrutura básica de pastas e arquivos para começar a desenvolver em React.

Props são utilizados para passar dados de um componente pai para um componente filho. State é gerenciado internamente pelo componente e pode mudar ao longo do tempo.
